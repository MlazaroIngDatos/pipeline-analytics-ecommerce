# 🚀 End-to-End Analytics Pipeline: E-commerce Perú (Medallion Architecture)

Este proyecto implementa un pipeline de datos moderno e híbrido para el procesamiento distribuido e ingesta analítica de transacciones de comercio electrónico en múltiples regiones del Perú. La solución utiliza el marco de trabajo **Medallion (Bronze/Silver/Gold)** sobre tecnologías de Big Data y almacenamiento relacional local.

---

## 🛠️ Arquitectura y Tecnologías

* **Procesamiento Distribuido**: Databricks Community Edition (Apache Spark & SparkSQL).
* **Almacenamiento e Ingesta Relacional**: SQL Server Local (Instancia `SQLEXPRESS02`).
* **Lenguajes y Herramientas**: Python 3.14, T-SQL, Visual Studio Code.
* **Librerías Clave**: `pandas`, `pyodbc`, `PySpark`.
